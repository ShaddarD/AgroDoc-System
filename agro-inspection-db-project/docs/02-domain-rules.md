# Domain Rules

## Для одной заявки
- один отправитель на русском
- одна доверенность отправителя (опционально)
- один получатель
- один продукт
- одно значение года урожая
- одно значение даты выработки
- одно назначение продукции
- один тип упаковки
- одна страна ввоза
- один представитель получателя
- одно место отбора
- одна лаборатория
- один номер контракта и одна дата контракта

## Для одной заявки может быть много
- контейнеров
- сертификатов
- регламентов

## Источник данных
- номер заявки — ручной ввод
- отправитель на английском — ручной ввод
- получатель — из справочника
- продукт на английском — ручной ввод
- порт выгрузки RU/EN — ручной ввод
- контракт — ручной ввод
