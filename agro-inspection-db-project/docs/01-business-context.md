# Business Context

Заявка — это основной документ системы. Она объединяет данные об отправителе, получателе, продукции, контракте, месте отбора, лаборатории, контейнерах, сертификатах и регламентах.

## Пользователи системы
- операторы, оформляющие заявки
- сотрудники, ведущие инспекцию
- администраторы справочников

## Почему нужны справочники
Справочники уменьшают дублирование данных, упрощают выбор типовых значений и обеспечивают единообразие записей.

## Почему номер заявки не PK
Номер заявки — удобный рабочий идентификатор для пользователя, но он не должен быть техническим ключом таблицы. Для внутренних связей используется UUID.
