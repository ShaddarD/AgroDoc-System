# API Concept

## Стиль
- REST API
- JSON
- UUID в URL
- поиск по номеру заявки отдельным endpoint

## Общая логика
- заявка создается как `draft`
- основные данные заявки хранятся в `applications`
- контейнеры, сертификаты и регламенты обслуживаются отдельными endpoint'ами
- справочники доступны через отдельный префикс `/api/dictionaries`
