# Form Workflow

1. Пользователь создает черновик заявки.
2. Заполняет основные поля заявки.
3. Выбирает отправителя, получателя, продукт, лабораторию и другие справочники.
4. Добавляет контейнеры.
5. Добавляет сертификаты.
6. Добавляет регламенты.
7. Проверяет обязательные поля.
8. Переводит заявку в статус `in_progress`.
9. После завершения заявка может перейти в `done`.
