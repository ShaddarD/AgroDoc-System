# Validation Rules

## Обязательные проверки
- `application_number` уникален
- обязательные поля не пусты
- `weight_mt > 0`
- `harvest_year` в разумном диапазоне
- `contract_date_manual` заполнена
- `planned_inspection_date` заполнена
- контейнеры не дублируются в пределах одной заявки

## Логические проверки
- выбранные UUID существуют в справочниках
- доверенность принадлежит выбранному отправителю
- статус меняется только по допустимым переходам
