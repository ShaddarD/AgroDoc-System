# Data Types Guidelines

## Рекомендуемые типы PostgreSQL
- UUID — технические ключи и внешние ключи
- TEXT — строки, номера, адреса, названия
- DATE — контракт, инспекция, доверенность, дата выработки
- TIMESTAMP — created_at, updated_at
- NUMERIC — вес партии
- BOOLEAN — флаги активности и необходимости
- INTEGER — год урожая, сортировка, количество копий
