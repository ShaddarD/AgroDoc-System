# application_certificates

## Назначение
Сертификаты, выбранные для заявки.

## Тип сущности
Дочерняя таблица.

## Поля
| Поле | Тип | Обязательное | Источник | Описание |
|---|---|---|---|---|
| id | UUID | да | система | PK |
| application_id | UUID | да | FK | заявка |
| certificate_id | UUID | да | справочник | сертификат |
| copies_count | INTEGER | нет | ручной ввод | число копий |
| is_required | BOOLEAN | нет | система | обязательность |
