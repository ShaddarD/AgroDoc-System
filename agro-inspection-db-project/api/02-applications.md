# Applications API

- `POST /api/applications` — создать заявку
- `GET /api/applications` — список заявок
- `GET /api/applications/{id}` — карточка заявки
- `PUT /api/applications/{id}` — полное обновление
- `PATCH /api/applications/{id}` — частичное обновление
- `POST /api/applications/{id}/change-status` — смена статуса
- `GET /api/applications/by-number/{application_number}` — поиск по номеру заявки
