🚀 README (Product + Portfolio Version)
🌾 AgroDoc-System

Платформа для автоматизации сертификации и экспортных операций сельскохозяйственной продукции

🎯 Зачем этот проект

AgroDoc-System решает типичную проблему отрасли:

❌ ручное оформление заявок
❌ Excel-таблицы и хаос в документах
❌ отсутствие контроля доступа
❌ сложный документооборот

👉 Система переводит всё в единый цифровой процесс.

🧠 Как устроен проект
Backend (Django)
applications → вся логика
reference → справочники
accounts → пользователи

👉 правило:

если сомневаешься — ищи в applications

Frontend (React)
api/ → все запросы
types/ → модели
pages/ → страницы
components/ → UI
🔥 Частые задачи
Добавить поле в заявку
models → serializer → frontend form
Добавить справочник
модель в reference
ViewSet
добавить в ReferenceTable
Ограничить доступ
backend → permissions
frontend → скрыть роут
⚠️ Важные правила
не ломать существующие API
не дублировать модели
использовать существующие паттерны
не писать логику во frontend
